API Dokümantasyonu
VoiceFlow Studio REST API — AI dublaj, altyazı ve ses işleme için tam entegrasyon
Base URL https://voiceflow-studio-production-395d.up.railway.app

Tüm endpoint'ler JSON veya multipart/form-data kabul eder. Yanıtlar her zaman JSON formatındadır. API, async işlem modeliyle çalışır — işlem başlatıldıktan sonra durum polling ile takip edilir.

Kimlik Doğrulama

Şu anki versiyonda API key gerektirmez — tüm endpoint'ler açıktır. Enterprise plan için header tabanlı auth eklenebilir.

Enterprise planında X-API-Key header'ı ile kimlik doğrulama eklenecek.

İstek Formatı

curl -X POST https://voiceflow-studio-production-395d.up.railway.app/api/islem/ \
  -F "modul=altyazi" \
  -F "kaynak_dil=tr" \
  -F "dosya=@video.mp4"

Hata Kodları

KodAçıklama

Hatalı yanıtlar {"hata": "açıklama"} formatındadır.

İşlem Başlat

POST /api/islem/ Video/ses işleme başlatır

Asenkron işlem başlatır. Dönen beklenen_dosya_adi ile durum sorgulanır.

Parametreler (multipart/form-data)

ParametreTipZorunluAçıklama
modulstringzorunludesifre | altyazi | seslendirme | metinden_sese
dosyafilezorunluVideo/ses dosyası (max 500MB)
kaynak_dilstringopsiyoneltr, en, de, fr, es... (varsayılan: tr)
hedef_dilstringopsiyonelÇeviri için hedef dil kodu
ses_idstringopsiyonelElevenLabs ses ID'si (dublaj için)
f_namestringopsiyonelFont adı (altyazı için, varsayılan: Arial)
f_sizestringopsiyonelFont boyutu px (varsayılan: 22)
f_colorstringopsiyonelHEX renk (varsayılan: #ffffff)
Örnek İstek (Python)
import requests

with open('video.mp4', 'rb') as f:
    r = requests.post(
        'https://voiceflow-studio-production-395d.up.railway.app/api/islem/',
        data={
            'modul': 'altyazi',
            'kaynak_dil': 'tr',
            'hedef_dil': 'en',
        },
        files={'dosya': f}
    )

print(r.json())  # {'beklenen_dosya_adi': 'sonuc_abc123.mp4'}
Yanıt
{
  "beklenen_dosya_adi": "sonuc_abc12345.mp4"
}

Durum Sorgula

GET /durum/{dosya_adi} İşlem durumunu döner

1-2 saniyede bir poll edin. yuzde: 100 olunca tamamlandı, yuzde: 0 + "Hata:" ile başlıyorsa hata var.

Örnek Polling (Python)
import time, requests

dosya_adi = "sonuc_abc12345.mp4"
base = "https://voiceflow-studio-production-395d.up.railway.app"

while True:
    r = requests.get(f"{base}/durum/{dosya_adi}")
    d = r.json()
    print(d['durum'], d['yuzde'])
    if d['yuzde'] == 100: break
    if d['yuzde'] == 0 and d['durum'].startswith('Hata'):
        raise Exception(d['durum'])
    time.sleep(2)

# Tamamlandı — indir
r = requests.get(f"{base}/indir/{dosya_adi}")
with open("output.mp4", "wb") as f:
    f.write(r.content)
Yanıt
{
  "durum": "Sesler üretiliyor (8/12)...",
  "yuzde": 62
}

Ses Klonlama

POST /api/ses_klonla/ Instant voice cloning

ElevenLabs Instant Voice Cloning ile ses klonlar. Creator+ plan gerektirir. Min 30s, max 25MB ses dosyası önerilir.

ParametreTipZorunluAçıklama
ses_dosyasifilezorunluMP3/WAV/OGG (max 25MB, min 30s önerilir)
isimstringopsiyonelKlonun adı (varsayılan: "Klonlanan Ses")

  "isim": "Benim Sesim",

}
